# 環境構築



## jQueryを開発するには一般的に下記のツールが必要となる



* **jQueryのライブラリ**



jQueryを動作させるのに必須なもの、これなきゃ始まらない。

http://jquery.com/download/

* Download the compressed, production jQuery 〇.〇.〇
* Download the uncompressed, development jQuery 〇.〇.〇

上記のどちらかをダウンロード、魔改造したいなら下のほうだがしないなら上のが軽い。

注：ライブラリをダウンロードせずにCDNを利用して外部サイトのjQueryを読み込む方法もある。


* **テキストエディタ**


Visual Studio Codeでも、Atomでも。

* **HTMLファイル**

母体となるもの、とりあえず適当につくる

## jQueryの使い方

CSSと同じように、HTMLファイルに記述しても良いしjQueryを外部に作って読み込んでも良い。
